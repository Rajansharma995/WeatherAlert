About WeatherAlert

WeatherAlert is a prototype web application created for the ICT726 Web Development assignment. It focuses on clean design, accessibility and simple, live weather data.

Our Purpose

The goal of WeatherAlert is to present complex weather information in a clear and friendly layout. The home page highlights key features, live city search and popular locations so users can quickly scan current conditions.

Technologies Used

The site is built using semantic HTML5, CSS3 and vanilla JavaScript. Live weather data is retrieved from the OpenWeatherMap API using the Fetch API. Responsive design is implemented using flexbox, CSS grid and media queries.

Users

WeatherAlert is designed for commuters, travellers and anyone who needs a quick snapshot of the weather before making decisions about their day. The interface is intentionally simple and accessible.
